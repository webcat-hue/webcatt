# webcatt
一、功能简介
 
本工具为纯前端本地运行的TOTP动态验证码生成器，遵循 RFC6238 标准，算法与Google Authenticator、Microsoft Authenticator通用。
所有密钥数据仅存储在浏览器本地 localStorage，不会上传至任何服务器，无需联网即可生成验证码。
支持多账号密钥管理、30秒自动刷新6位动态码、倒计时提示、密钥删除功能。

二、使用步骤
 
1. 获取平台Base32密钥
进入需要两步验证的网站或后台，开启二次验证，选择手动输入密钥，复制页面给出的Base32字符串。
​
2. 添加账号信息
账号名称填写平台名称，例如GitHub、Steam，方便区分不同账号；Base32密钥粘贴复制好的密钥文本，点击添加按钮保存。
​
3. 查看动态验证码
点击密钥列表内任意账号条目，页面上方会展示当前周期6位验证码，下方倒计时显示该验证码剩余有效时长。
​
4. 删除密钥
点击对应账号右侧红色删除按钮，即可移除该账号保存的密钥。
 
三、注意事项
 
1. 清除浏览器缓存、切换浏览器、无痕模式关闭后，本地保存的密钥会全部丢失，重要密钥建议自行备份记录。
​
2. 仅支持标准Base32密钥，包含数字0、1、8、9的密钥无法正常解码使用。
​
3. 验证码每30秒刷新一次，过期验证码无法用于登录验证，使用时请确认倒计时未结束。
